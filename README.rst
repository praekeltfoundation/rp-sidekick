=================
Rapidpro Sidekick
=================

Sidekick app for Rapidpro.

This app is used to extend the functionality of Rapidpro.


Local installation
------------------

    $ git clone git@github.com:praekeltfoundation/rp-sidekick.git

    $ cd rp-sidekick

    $ virtualenv ve

    $ source ve/bin/activate

    $ pip install -e .
